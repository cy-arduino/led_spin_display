# led_spin_display


## usage
1. connect to HC-05(BT SPP)
1. use UART to config
	- example: gogogo.txt

- demo video: https://www.youtube.com/watch?v=6d0FzNaL9l4&feature=youtu.be


---

- 載具(開發用):
          - 馬達: 無刷( 2212, 980KV)
          - 電變: 30A
          - PWM轉速控制: arduino nano
    
- 本體
          - LED:
          - 單色16顆
    
- 供電:
          - 100mA鋰電池
    
- 傳輸:
          - 無線(BT HS-05, SPP)
    
- 轉速偵測:
          - 光遮斷器
    
- 開發版:
          - arduino nano
